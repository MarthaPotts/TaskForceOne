# TaskForceOne

## Description 

TaskForceOne is an Employee Management System operable directly from the command line, offering several means of access to your organized employee data in one convenient location. As your business expands, TaskForceOne can expand with your growing needs.  

## Installation

Download the source code from the repository. This is a Node.js/npm/mysql dependent program and will not run without having either of those installed. 

With the prequisite dependencies installed invoke the program with 'node index.js'. TaskForceOne should load in your console.
